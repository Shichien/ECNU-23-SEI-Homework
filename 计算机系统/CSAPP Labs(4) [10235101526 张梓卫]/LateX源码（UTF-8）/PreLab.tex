\begin{itemize}
    \item 解压 valgrind 包：tar -jxvf valgrind-3.19.0.tar.bz2
    \item 进入 valgrind 目录：cd valgrind-3.19.0
    \item 初始化配置： ./configure
    \item 编译并安装：make / \& / sudo make install
    \item 检查是否安装成功：valgrind–h 显示 valgrind 的参数及提示，说明安装成功
\end{itemize}
